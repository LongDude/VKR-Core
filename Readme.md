# Описание
В конечной конфигурации система ScInside будет состоять из 3х основных микросервисов:
- Frontend - Клиент
- Backend - обработка запросов клиента, выполнение запросов к БД
- MLService - Популяция базы данных новыми записями, сложные запросы вроде "анализа предметной области"

Metacluster состоит из всех сервисов и настроек, необходимых для совместного функционирования основных сервисов:
- Объявление внутренней и внешней сети
- Общая база данных (PostgreSQL) и клиент для ее модерации (PgAdmin)
- Обратный прокси Nginx ( в будущем )
